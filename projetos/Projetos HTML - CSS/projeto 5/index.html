Principais formatações

negrito / destaque

Para colocar uma palavra em negrito usa a tag b. (não tem semantica)

Para colocar a palavra em destaque usa a tag strong. (tem semantica)

Para itálico é usado <i> (não semantica) ou <em> (semantica)

Envelopar

teste de envelopamento (colocar a tag dps de feito a frase)

Marcar

Para marcar como um marca texto usa a tag <mark>.

Grande ou pequeno

Para deixar a letra grande usa a tag <nig>. e para deixar menor usa a tag <small>.

Texto deletado

Podemos marcar um texto deletado com a tag <del>

Texto insirido

Para inserir um texto é usado a tag <ins>

Texto sobrescrito

Para mostrar contas com potências como x20+3 usa a tag <sup> (para superior)

Texto subscrito

Para mostrar o numero na parte de baixo como H2O usa a tag <sub>. (para inferior)
